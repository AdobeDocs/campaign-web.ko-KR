# 다이내믹 콘텐츠 시작

콘텐츠를 동적으로 만드는 방법:
- 다양한 출처의 속성으로 콘텐츠 개인화
- 대상에 맞게 조정되는 조건부 콘텐츠 만들기

중요한 이유, 이점은 무엇입니까, 사용 예

개인화할 수 있는 사항 : 표현식 편집기를 사용한 모든 게재

액세스 방법 :
- perso 아이콘이 있는 모든 필드(제목 줄, sms 또는 푸시 본문...)-
- 이메일 디자이너
